# ResearchMate AI ✦
**The 24-Hour Innovation for Academic Efficiency**

ResearchMate AI is an intelligent agent that transforms raw research notes, rough drafts, or unstructured PDFs into fully formatted, conference-ready research papers. By automating the tedious parts of academic writing—formatting, table structuring, and image placement—it allows researchers to focus entirely on their core ideas.

### 💡 The Problem
Students and researchers spend nearly **40% of their time** struggling with LaTeX/Word formatting, IEEE styling, and aligning tables/images instead of actually researching.

---

### ✨ Our Solution
ResearchMate AI provides a three-step automated pipeline:

1. **Extraction:** High-accuracy PDF parsing that pulls text and preserves complex visual data like charts and images.  
2. **AI Intelligence:** A custom agentic workflow powered by **Groq (Llama 3)** that structures content into academic sections (Abstract, Methodology, Results).  
3. **Multi-Format Export:** Instant generation of pixel-perfect **IEEE PDFs** (server-side via ReportLab) and editable **DOCX** files (generated locally in the browser for speed).  

---

### 🛠️ Tech Stack
* **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Lucide Icons  
* **Backend:** FastAPI (Python 3.10), Groq API, ReportLab, PyPDF, SQLAlchemy  
* **Deployment:** Vercel & Render
